#### Axes construction
```
fig.add_axes(left, bottom, width, height)
```

#### Item hiding
1.Hide axis (both ticks and tick labels)
```
ax.set_visible(False)
ax.set_visible(False)
```
2. Hide ticks
```
ax.set_xticks([])
ax.set_yticks([])
```
3. Hide tick labels
```
ax.set_xticklabels([])
ax.set_yticklabels([])
```
